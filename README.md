# Trabalho de GitHub

## 📌 Descrição
Este trabalho tem como objetivo aplicar, na prática, os conceitos de versionamento de código utilizando o GitHub. Foram utilizados recursos como criação de repositórios, branches, commits e merge, além da utilização de arquivos no formato Markdown (.md).

## 👥 Integrantes

## 📂 Estrutura do Projeto
Cada integrante desenvolveu um arquivo individual em sua própria branch, contendo informações pessoais e aprendizados adquiridos. Posteriormente, os arquivos foram integrados ao repositório principal por meio de merge.

## 🚀 Tecnologias utilizadas
- Git
- GitHub
- Markdown
