# Fundamentos de Git e GitHub
 Olá, meu nome é Vitor Miguel Rocha Calheiros e explicarei sobre os fundamentos de Git e Github.
## O que é Git?
Git é um sistema de controle de versão distribuído, utilizado para gerenciar o histórico de alterações em projetos.

## O que é GitHub?
GitHub é uma plataforma online que hospeda repositórios Git, permitindo colaboração entre desenvolvedores.

## O que é um repositório?
É o local onde o projeto e seu histórico de versões são armazenados.

## O que é commit?
Commit é o registro de uma alteração no projeto. Cada commit possui uma mensagem descritiva.

## O que é branch?
Branch é uma ramificação do projeto principal, usada para desenvolver funcionalidades sem afetar a versão principal.

## O que é merge?
Merge é o processo de unir alterações de uma branch em outra, geralmente para integrar na branch principal (main).
